# szsk前端公共函数库
    + 各种类型下的工具方法

## 使用范围
通用

## 使用方式
npm i @szsk/utils

import { funcUtils } from '@szsk/utils';

ts使用
```
  // tsconfig.json
  "compilerOptions": {
    "typeRoots": [
      // 加入这个进行自动提示
      "node_modules/@szsk",
    ]
  },
```
